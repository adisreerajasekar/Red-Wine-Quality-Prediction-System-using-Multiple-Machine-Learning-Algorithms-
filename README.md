# Red Wine Quality Prediction System Using Multiple Machine Learning Algorithms

## Overview
This project focuses on predicting the quality of red wine using multiple Machine Learning algorithms. The system analyzes physicochemical properties of wine and predicts its quality score based on the given input features.

The project includes data preprocessing, exploratory data analysis, visualization, model training, and comparison of different machine learning algorithms.

---

## Dataset
The dataset used is the Red Wine Quality dataset, commonly used for wine quality prediction tasks.

### Dataset Features
The dataset contains physicochemical properties such as:

- Fixed Acidity
- Volatile Acidity
- Citric Acid
- Residual Sugar
- Chlorides
- Free Sulfur Dioxide
- Total Sulfur Dioxide
- Density
- pH
- Sulphates
- Alcohol
- Quality

The target variable is:

- `quality` → Wine quality score

---

## Problem Statement
Wine quality evaluation is usually performed through human tasting, which can be subjective and time-consuming. This project applies machine learning techniques to predict wine quality using measurable chemical properties.

The goal is to build models that can classify or predict wine quality accurately based on input features.

---

## Objectives
- Analyze the red wine quality dataset
- Perform data cleaning and preprocessing
